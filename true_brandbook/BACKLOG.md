# Design Terminal — Backlog

Все отложенные задачи, идеи и решения которые не вошли в MVP.

---

## UX / Интерфейс

- Fun статусы генерации пока идёт ожидание (типа "замешиваем пиксели...")
- Браузерные push-уведомления когда генерация готова (пользователь ушёл со страницы)
- Настройка приватности activity feed (некоторые компании не хотят показывать кто что делает)
- Мобильная оптимизация формы генерации

## Брендбук

- Чат-интерфейс для мастерадмина — настройка брендбука через разговор с Claude
- Версионирование брендбука (история изменений)
- Поддержка нескольких суббрендов в одном workspace

## Генерация

- Перегенерация отдельных форматов без пересоздания всего батча
- Улучшенная валидация HTML (CSS lint, проверка использования брендовых цветов)
- Очередь генерации при высокой нагрузке

## Аналитика и администрирование

- Дашборд потребления с графиками по форматам, странам, времени
- Уведомление workspace admin когда близко к лимиту
- Экспорт статистики

## Инфраструктура

- Мульти-workspace для одного пользователя
- Race condition лимитов под высокую нагрузку (сейчас достаточно для MVP)
- CDN для хранения финальных файлов (при росте объёма)

## Интернационализация

- Поддержка дополнительных языков интерфейса (DE, PL, CS и др.)
- Автодетект языка по браузеру

## Монетизация (будущий SaaS)

- Тарифные планы (Free / Pro / Enterprise)
- Биллинг через Stripe
- Онбординг для новых workspace
- Лендинг страница сервиса
