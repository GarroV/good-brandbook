# Design Terminal — Master Project

## Суть проекта

Веб-сервис для генерации маркетинговых макетов по брендбуку. Маркетолог пишет текстом что нужно, опционально прикрепляет референс, выбирает форматы — система генерирует готовые файлы строго по правилам бренда.

Готовые макеты публикуются в общую галерею. Любой маркетолог из любой страны может взять чужой макет за основу и адаптировать под свой запрос.

---

## Проблема которую решаем

Маркетологи в странах регулярно делают одно и то же: вручную верстают макеты под разные форматы, часто нарушая брендбук. Нет единой библиотеки готовых материалов. Нет инструмента который знает бренд и генерирует по правилам.

---

## Целевой пользователь

Маркетолог в стране-партнёре. Не дизайнер. Знает что нужно сделать, не знает как сверстать. Работает в браузере, может быть на телефоне.

---

## Ключевые сценарии

1. Маркетолог открывает сервис, выбирает форматы, пишет запрос, жмёт "Сгенерировать"
2. Видит превью по мере готовности каждого формата
3. Одобряет — макет публикуется в общую галерею
4. Скачивает файл (PDF для печати, PNG для цифры)
5. Находит в галерее удачный макет другой страны, нажимает "Взять за основу", адаптирует

---

## Архитектурные принципы

- Мультитенантность с первого дня (workspace_id + RLS везде)
- Один пользователь = один workspace (MVP)
- Брендбук — центральный контекст каждой генерации
- Без заполненного брендбука генерация заблокирована
- Черновики хранятся 3 дня, опубликованные — навсегда
- Один батч с N форматами = 1 генерация по лимитам
- Двухэтапный рендер: сначала JPEG превью, финальные файлы после одобрения

## Стек

| Слой | Технология |
|------|-----------|
| Фронтенд + API | Next.js (App Router) |
| База данных | Supabase (Postgres + RLS) |
| Хранилище файлов | Supabase Storage |
| Аутентификация | Supabase Auth |
| Realtime | Supabase Realtime |
| AI генерация | Claude API (claude-sonnet-4-6) |
| Рендер файлов | Puppeteer + Chromium |
| Email | Resend |
| Деплой | Railway |
| UI компоненты | shadcn/ui + Tailwind CSS |
| i18n | next-intl (EN + RU, архитектура под расширение) |

## Роли

| Роль | Возможности |
|------|------------|
| `superadmin` | Все workspace, лимиты, создание workspace |
| `admin` | Свой workspace, приглашения, брендбук, удаление макетов |
| `member` | Генерация, просмотр галереи, скачивание |
